Multi-platform OBD Scanner

Mobile App

A multi-platform On-Board Diagnostics (OBD) scanner developed as part of my final year project.

The project was deployed as both a Windows 10 Universal App and Android app.

This was achieved through using the Xamarin framework, allowing all code to be written in C#, making a completely portable system.

This project was a runner up in ActionPoint's Most Commercially Viable Final Year Project award
